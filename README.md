# Unidad 2 - Proyecto de Programación

## Desarrollo Comunitario 01 - Timed-Generator

Este es el primer proyecto de Desarrollo Comunitario, enmarcado en nuestro Proyecto de Programación.
Logramos desarrollar una clase base `Generador`, y en base a ella implementamos un `GeneradorUniforme`.

Existe un ejemplo de otras clases necesarias en el proyecto, y cómo se vinculan para lograr la generación de clientes.

## Actividad con décimas

De acuerdo con nuestra implementación, existe un insentivo de **3 décimas** para la Entrega Final del proyecto.

Para ello, las instrucciones son:
1. Deben crear un fork del proyecto comunitario.
1. En su propio fork, deben desarrollar otra implementación de un `Generador`, que funcione correctamente y sea utilizado al igual que lo hacemos con el `GeneradorUniforme`.
1. Deben crear un pull-request hacia el Repositorio Comunitario, lo que permitirá que todos los equipos del curso tengan acceso a su nuevo generador.
